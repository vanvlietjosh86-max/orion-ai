# ORION Development Roadmap

## Simple enough to understand. Smart enough to trust.

Version: 1.0

---

# Overview

The ORION roadmap defines the planned evolution of the Intelligent Investing Companion.

Development will happen in stages, with each version adding useful capabilities while maintaining simplicity, transparency, and reliability.

---

# Version 0.1 — Foundation

## Goal

Create the first working ORION experience.

Features:

- Project architecture.
- User profile.
- ORION personality system.
- Home dashboard.
- Settings.
- Learn Mode.
- Local data storage.
- Basic AI interaction.

Status:

In development.

---

# Version 0.2 — Market Intelligence

## Goal

Introduce market awareness.

Features:

- Stock search.
- ASX market focus.
- Watchlists.
- Company information.
- ETF information.
- Market summaries.

---

# Version 0.3 — Paper Trading

## Goal

Allow learning without financial risk.

Features:

- Performance tracking.
- Investment journal.
- Decision history.

# Version 0.4 — Portfolio Intelligence

## Goal

# Future Expansion

Possible future modules:

- Dividend tracking.
- Tax awareness.
- Superannuation planning.
- Retirement projections.
- Additional asset classes.
- Advanced analytics.

---

# Development Philosophy

ORION will always prioritise:

1. Reliability over speed.
2. Understanding over complexity.
3. Long-term investing over speculation.
4. Education over blind recommendations.
5. User control over automation.
